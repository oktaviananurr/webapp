Project ini adalah aplikasi web berbasis Streamlit yang dibuat oleh Kelompok 1 Departemen Statistika Bisnis ITS untuk menampilkan informasi akademik, khususnya jadwal seminar proposal proyek akhir mahasiswa semester gasal 2023/2024, sekaligus memperkenalkan tim pengembang.

**Teknologi yang Digunakan**
Streamlit → Framework Python untuk membuat aplikasi web data secara cepat.
Bootstrap Icons → Untuk mempercantik tampilan dengan ikon.
use_container_width → Parameter layout terbaru yang menggantikan use_column_width
